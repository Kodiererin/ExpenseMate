## 🚀 **How To Use New Features**

### For Users:

**Track Income:**
1. Tap "Add" tab
2. Toggle to "Income"
3. Select income source
4. Enter amount
5. Tap "Add Income"

**Chat with AI:**
1. Tap "AI Chat" tab (with NEW badge)
2. Use quick actions OR type question
3. Get instant insights
4. Continue conversation

**Set Recurring:**
1. In Add screen
2. Toggle "Recurring Transaction"
3. Select frequency
4. Add as normal

**Export Data:**
1. Go to Analytics
2. Tap export button (↓)
3. Choose CSV or Summary
4. Share via email/cloud

---

### For Developers:

**Add AI Integration:**
1. Get API key from provider
2. Update `constants/api.ts`
3. Replace `generateAIResponse` in `ai-chat.tsx`
4. Test with sample queries
5. Deploy!

**Customize Categories:**
1. Edit `constants/categories.ts` or `constants/income.ts`
2. Add new category with icon and color
3. Categories auto-populate in dropdowns

**Add New Features:**
1. Follow existing patterns
2. Use TypeScript types
3. Maintain accessibility
4. Update documentation
